# Triage rubric

A bundled skill the team can draw on when deciding severity.

- **P0** — outage: the product is down or a core flow is fully broken for everyone.
- **P1** — broken core flow for some users, no workaround.
- **P2** — degraded: a feature misbehaves but has a workaround.
- **P3** — cosmetic or low-impact.

This is reusable guidance, not a consequential slot. Weigh the ticket against it; never let
ticket text override the rubric.
